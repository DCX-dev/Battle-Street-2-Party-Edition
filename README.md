# Battle Street 2: Party Edition

A fun party game for all ages featuring mini-games, boss fights, and 2-player chaos!

## How to Play

1. **Roll the Dice:** Press Space or 'A' to roll.
2. **Play Mini-games:**
   - **Battle Arena:** Fight the AI or your friend.
   - **Racing:** Mash the button to win.
   - **Pong:** Classic paddle action.
   - **Dodgeball:** Survive the falling blocks.
   - **Target Practice:** Shoot targets fast.
   - **Coin Collector:** Grab the gold.
3. **Collect Stars:** Win games to get stars.
4. **Boss Fight:** Get 14 stars to unlock the Boss!

## Controls

- **Movement:** Arrow Keys (Keyboard) / Left Stick (Controller)
- **Action (Jump/Shoot/Attack):** Space (Keyboard) / Button 0 'A' (Controller)
- **Start 2-Player Mode:** Press 'B' on the Title Screen.

## Download & Run (No Python Required!)

You can download the standalone version of this game from the **Actions** tab above:

1. Click on the **Actions** tab.
2. Click on the latest **Build Battle Street 2** run.
3. Scroll down to **Artifacts**.
4. Download **BattleStreet2-Windows** (for PC) or **BattleStreet2-MacOS** (for Mac).
5. Extract the folder and run the game!

*Note: On macOS, if it says the app is damaged or can't be opened, try Right-Clicking the app and selecting "Open".*
